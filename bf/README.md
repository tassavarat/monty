### [Brainf*ck]
Write a Brainf*ck script that prints Holberton, followed by a new line.
```julien@ubuntu:~/bf$ bf 1000-holberton.bf 
Holberton
julien@ubuntu:~/bf$
```
##### [add 2 digit]

The total of the two digits with be one digit-long (<10)
```julien@ubuntu:~/bf$ bf ./1001-add.bf
81
9julien@ubuntu:~/bf$
```
